Node.js with Express
-----------------------
	Express.js is a minimal and flexible Node.js web application framework for building single-page,
	multi-page, and hybrid web apps.

Standard backend framework in MEAN stack:
--------------------------------------
	MEAN Stack:
	MongoDB – NoSQL Database
	Express.js – Web framework
	Angular – Frontend MVC
	Node.js – Runtime


getting started:
--------------
npm init -y
npm install express
